Just a dumb minesweeper made under 6h.

# How to run:
compile everything and run Main.java

# How to start:
1. Choose the board size (the board is fixed to squares only lolol... for now)
2. Choose the difficulty
3. Now you can play

# How to play:
1. You can two options
  - You can enter in the coordinate to interact with the tiles
  - ( you must enter the coordinate in `x y` format || e.g: `6 7` )
  - You can change the mode by entering "flag" or "dig"
2. Dig Mode (Default)
  - You will be digging, better hope it's not a mine~
3. Flag Mode
  - You will be enable or diabling flag
4. Other are basic minesweeper rules.
5. HAVE FUN
